## 速速投

《速速投》项目主要是一个快速投递简历的平台，通过一份简历利用训练好的大模型和算法推荐职位列表，后续考虑推出AI面试官功能；



#### 主要技术流程分析

主要技术栈：Langchain | Mysql | Vue | 

流程：用户上传一份简历(.pdf)将其交给Ai分析，分析其职位然后通过mcp搜索+爬虫获取市面上各大厂商的职位及其JD，如果用户想要投递对应职位，可以通过一键优化简历，可以直达官网投递链接

难点：如何检索各大厂商官网的职位信息？有新的厂商更新网站如何添加到我的数据库？现有的厂商列表如何快速添加？爬取Boss直聘,猎聘等网站的信息？
数据获取计划：
1. 先从boss直聘等网站爬取职位信息，保存到数据库中
2. 当有新的厂商时，手动添加其网址到数据库中
3. 每隔一定时间定时爬取厂商的职位信息，更新数据库

对话逻辑：
1. 用户上传一份简历(.pdf)将其交给Ai分析，分析其职位然后通过mcp检索数据库中的职位信息(根据职位名称和工作地点,可以深入JD信息进行匹配)，然后接口返回数据字段：position.id,position.name,position.location,position.jd,position.salary,position.link
2. 用户可以点击一键优化简历，优化后的简历会直接下载到本地
3. 用户可以点击投递链接，直接跳转到对应职位的官网投递页面(position.link)
